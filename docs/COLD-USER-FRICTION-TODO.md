## 여정 지도 (문제 위치)

```
/  →  온보딩 게이트(DashboardPage.tsx:263)
   →  면죄부 3화면 (OnboardingPage.tsx:15)      ← 감정만 가르침, 작동법 0
   →  빈 대시보드 + 입력 (AddTaskPrompt)          ← [C1] 스펙은 "블록 낙하"인데 빈 상태
   →  쪼개기 (SplitPage)                          ← [A1][A2] 이게 뭐지 병목
   →  대시보드 복귀
   →  예측 (PredictPage)                          ← 제스처 힌트 1회성(여기서만)
   →  타이머 (FocusPage)                          ← [B1][B2] 보이는 조작 0개, 이탈 안내 없음
   →  마무리/회고
```

**핵심 진단:** 온보딩 3화면은 면죄부(감정)만 가르치고 작동법은 전부 실화면 just-in-time. 그런데 쪼개기·타이머 두 화면이 설명 부족. → 콜드 유저 마찰의 뿌리.
