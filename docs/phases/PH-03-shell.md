## Goal

React Router 라우트 골격 + 앱 셸 + PWA manifest/SW 등록이 완성되어, 각 라우트가 (내용 없어도) 최소 동작 placeholder를 실제로 렌더하는 상태. 주석 처리된 미완성 코드 금지 — placeholder는 실제 컴파일·렌더되는 컴포넌트여야 한다.
