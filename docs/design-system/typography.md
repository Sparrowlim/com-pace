# 타이포 위계 (Typography) — `TY`

> 상위: [`../DESIGN-SYSTEM.md`](../DESIGN-SYSTEM.md) 허브 · 값 정본: [`../DESIGN-TOKENS.md §2-8`](../DESIGN-TOKENS.md) (`font.*`)
> 위계 근거: `DESIGN-BRIEF DB-03`(서체 2종·세리프 제한) 재확인 — 신규 값 없음.

## TY-1. 동시 노출 크기 ≤ 3단계

카드 1개 또는 화면 1개 안에서 동시에 쓰는 `font.size` 조합은 **최대 3단계**(주·보조·라벨)로 제한한다.

| 단계        | 역할                       | 권장 토큰                      | 서체                               |
| ----------- | -------------------------- | ------------------------------ | ---------------------------------- |
| 주 (제목)   | 카드/화면의 대표 문구      | `font.size.xl`~`font.size.2xl` | `font.serif`(유일한 세리프 사용처) |
| 보조 (본문) | 설명·CTA 라벨              | `font.size.md`~`font.size.lg`  | `font.sans`                        |
| 라벨 (메타) | 힌트·필드 라벨·조용한 안내 | `font.size.xs`~`font.size.sm`  | `font.sans`                        |

한 카드에 이 3단계를 넘는 크기 조합이 등장하면 위계가 무너진 것으로 간주한다(예: 제목 2종·본문 2종을 동시에 쓰는 것 금지).

## TY-2. 세리프는 제목 1곳에만

`font.serif`는 **제목 1곳에만** 한정한다(`DB-03`). 본문·라벨·UI는 `font.sans`. 온보딩 헤드라인처럼 화면 전체를 대표하는 단일 문구에만 세리프를 허용한다.
